# novora_flutter

A new Flutter project.

## Run (important)

`pubspec.yaml` lives **in this folder**. From the **monorepo root** (`novora/`), run:

```bash
cd novora_flutter && flutter run -d chrome
```

Or from the **monorepo root** (parent of this folder): `./run_flutter.sh -d chrome`.

In **Cursor / VS Code** with the repo root open: **Run and Debug** → **Flutter: novora_flutter (Chrome)**.
